# 🛡️ SOC Analyst Lab

## Overview
This project demonstrates hands-on SOC analyst skills including threat detection, log analysis, and incident response using simulated scenarios.

## Tools Used
- SIEM (Splunk / ELK Stack)
- Wireshark
- Linux

## Skills Demonstrated
- Log analysis
- Threat detection
- Incident response
- Network investigation

## Project Structure
- screenshots/ → Evidence of alerts and analysis
- logs/ → Sample log data
- reports/ → Investigation report

## Outcome
Successfully identified and analyzed simulated threats, demonstrating real-world SOC analyst responsibilities.
